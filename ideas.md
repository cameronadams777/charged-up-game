# Charged Up Game

## Core game
[ ] amount of lives
  - start with 3
  - once you reach 0, game over
[ ] top down
  - moving map
  - no forward controls
[X] collect a charge
  - collect items
    - traffic cones - +5
    - inflatable cubes - +1
[ ] avoid opposing alliance bots
  - opposing alliance is red
  - my alliance is blue
[ ] last for 2 minutes

[ ] leaderboard
  - take a team number to record
  - reports charge count
  - Spinning up DB
  - Making API requests for the leaderboard
